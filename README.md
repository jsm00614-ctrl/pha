# PHA 기반 생분해성 화장품 용기 적용 검토

㈜다시물결(Dasiwave) REO 컴파운드 적용 가능성 및 구매조건부 국가과제(구매연계형 R&D) 연계 검토 보고.

## 파일 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 웹 보고서 (반응형) — GitHub Pages 진입점 |
| `PHA 검토 보고서 (웹).html` | 웹 보고서 (동일 파일) |
| `PHA 검토 보고서.html` | A4 인쇄/PDF용 7페이지 보고서 (Tweaks 컬러테마 포함) |
| `PHA 검토 보고서 (단독실행).html` | 폰트·스크립트 내장 오프라인 단독 실행본 |
| `tweaks-panel.jsx`, `tweaks-app.jsx` | A4 보고서 컬러테마 토글 스크립트 |
| `fonts/` | 임베드 폰트 |

## GitHub Pages 배포
저장소 Settings → Pages → Branch `main` / root 선택 후, `index.html`이 자동 게시됩니다.
